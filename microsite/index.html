Quick Start Tutorial

How to use GOVDAO

A visual guide on setting up your wallet, casting votes, submitting proposals, and monitoring treasury operations.

Connect Your Web3 Wallet

To interact with the on-chain GOVDAO platform, you need an EVM-compatible wallet like MetaMask.

A
Add Localhost Network: Configure your wallet to connect to http://127.0.0.1:8545 with Chain ID 31337 for testing.
B
Import Test Account: Use the private key 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80 to import a seeded account with mock ETH.
C
Connect Wallet: Click "Injected Wallet" in the app header, and approve the connection in MetaMask.

Cast Votes & On-Chain Roles

Your voting power is determined on-chain by the MemberRegistry. Connect your wallet to resolve your role automatically.

A
On-chain Roles: Access levels span Observer, Member (can vote), Proposer (can create proposals), Admin (manage roles), and Guardian.
B
Active Proposals: Select any proposal from the list view to inspect proposal text and vote distributions.
C
Cast Vote: Choose For, Against, or Abstain, click "Cast Vote", and confirm the transaction signature.

Creating Proposals

Proposers and Admins can draft and publish new proposals directly to the Governor smart contract.

A
Use Templates: Prefill formatting using categories like Treasury Spends, Member Invites, or Code Upgrades.
B
Submit Draft: Enter title, description, and target action data, then execute the proposal creation transaction.
C
Voting & Timelock: Once submitted, proposals move to "Proposed" (warmup), then become "Active" for voting. Success triggers a 24-hour Timelock delay.

Treasury & Emergency Safeguards

Govern the organization's funds securely and test fallback multi-sig emergency controls.

A
Treasury Allocations: View asset holdings, transaction limits, and pending funding requests in real-time.
B
Emergency Pause: If a vulnerability is found, the Guardian role can pause proposal lifecycle functions for up to 72 hours.
C
Guardian Drills: Admins can schedule simulated pause rehearsals to verify guardian reaction times.
